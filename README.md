# Shop-Stock
Web application(FE + BE) for placing orders and viewing available products.<br />

Backend built with TypeScript:<br />
-TypeORM<br />
-JWT Authentication<br />
-Express<br />
-class-validator<br />
-cors<br />
-bcrypt<br />
<br />
FrontEnd built with Vue.js(vite)<br />
-axios<br />
-bootstrap<br />
-lodash<br />
-vue-router<br />
-vue-sweetalert2<br />
